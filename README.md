# swe-testing-assignment

Orkhan's Caculator:
A command-line calculator built with python. This application was developed with a focus on testing, correct documentation, and good practice of version control.

Features:
- The logic and the UI are separated, which allows for independent testing of mathematical operations.
- User input is normalized where necessary (for example typing 'C' to clear) and unexecpected data is handled gracefully.
- There is a full test suite for unit and integration scenarios.

Prerequisites: Python 3.10 or higher

How to run:
1. Clone the repository: git clone https://github.com/oorkkaa/swe-testing-assignment
2. navigate to project folder: cd swe-testing-assignment
3. Run main.py

Command to run unit and integration tests:
pytest
